### 프로젝트 폴더 구성

* components - 컴포넌트 정의
* containers - 컨테이너 컴포넌트 정의
* pages - 페이지 컴포넌트 정의
* redux-modules - 리듀서와 액션을 정의해놓은 파일들
* util - 공통 기능이나 타입들을 정의해놓은 파일들

### 그 외

* Common.scss - 공통으로 쓰는 클래스나 값 정의
* 각 폴더별 index.ts 파일들 - 여기에 모두 import 후 export 합니다

### 기술스택

* React Hooks
* Typescript
* React-router-dom
* Redux
* emotion / SCSS
